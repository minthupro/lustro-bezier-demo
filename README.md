# Lustro Silk Easing Demo

A slideshow demo showcasing silk cubic-bezier CSS transitions from the
[Lustro TM-624 Slideshow Template](https://templatemo.com/tm-624-lustro-slideshow)
on TemplateMo.

## What this demonstrates

- Silk easing with `cubic-bezier(0.19, 1, 0.22, 1)`
- Staggered caption reveal using CSS transition-delay
- Vanilla JS slide switching — zero libraries
- Double `requestAnimationFrame` reflow trick for smooth re-entry
